This plugin adds Jitbit Live Chat widget to every page of your Wordpress blog.

== Description ==

To use this plugin you need to have a [Jitbit Live Chat](http://www.jitbit.com/livechat/) account. You can register for a free trial here: http://www.jitbit.com/livechat/trial/ It takes 30 seconds and does not require a credit card.

After the registration, you will get your own live chat URL (https://%your_company_name%.jitbit.com/chat), you will need it to make this plugin work.

Jitbit Live Chat for Wordpress loads an external javascript file from Jitbit's servers. It will execute in your visitors browsers, but don't worry, the only purpose of the plugin is to create a live chat widget.

Also please note that all communication between your Wordpress site and Jitbit servers is secured with SSL.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the "Jitbit Live Chat" menu and enter your full Live Chat URL like https://chat.jitbit.com/chat. Do not forget to include /chat on the end.

That's it. Now you should see the Live Chat widget on the  bottom of every public page of your web-site.
